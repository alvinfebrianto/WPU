Prototype

Properti: Variable yang ada di dalam object

Method: Function yang ada di dalam object

Constructor: Function yang khusus untuk membuat object

Secara konvensi, penulisan nama function pada constructor menggunakan PascalCase

This: Sebuah keyword special yang secara otomatis didefinisikan pada setiap function

Context: Setiap membuat variabel di scope global, artinya variabel ialah properti/method dari object window
